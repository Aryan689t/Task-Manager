# 🗂️ Task Manager Web App

A simple and interactive **Task Manager** built using **HTML, CSS, and JavaScript**.
This application allows users to create, edit, manage, and track tasks with persistent storage using **localStorage**.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit tasks (with edit control)
* ✅ Mark tasks as Complete / Incomplete
* 🗑️ Delete selected task
* 🧹 Clear input field
* 💾 Data persistence using localStorage
* 🎯 Highlight selected task
* 📌 New tasks appear at the top

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)
* Browser localStorage

---

## 🙌 Learning Outcomes

* DOM manipulation
* Event-driven programming
* State management
* Local storage handling
